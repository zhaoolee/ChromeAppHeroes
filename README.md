## 用完后, 记得退出setupvpn
- 开启插件后, 浏览器所有的请求都会走代理, 访问百度,淘宝等常用网站也会变得特别慢,所以不访问外网的时候, 记得关闭插件
> ![](https://upload-images.jianshu.io/upload_images/3203841-ac7b3e6d5c527547.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

>  ![](https://upload-images.jianshu.io/upload_images/3203841-b65a4e685983e3e3.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

#### 006《SimilarSites》 一键查找姊妹网站 SimilarSites
> ![](https://upload-images.jianshu.io/upload_images/3203841-1dcc29e97f4f7bba.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)
